# Nutrition System

Local startup is intentionally simple.

## Start The App

From the `app` folder, run:

```bash
php artisan serve
```

Then open `http://127.0.0.1:8000`.

## Notes

- Frontend assets are already built in `public/build`, so no `npm run dev` is required to run locally.
- If this is a fresh machine, install PHP dependencies once using `composer install` before running the command above.
